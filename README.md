# CypherX Bot 🤖

CypherX ni bot rahisi ya kuseti na kutumia kwenye majukwaa kama **Discord / Telegram** (kutegemea setup yako).  
Imeandaliwa kwa ajili ya automation, moderation na features nyingine za kijanja.
